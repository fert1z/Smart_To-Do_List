# Smart To-Do List

A web-based task management application designed to help users organize their daily tasks efficiently. Built with HTML, CSS, and potentially a backend for full functionality.

## Features

- **Task Management**: Create, edit, and delete tasks
- **User Authentication**: Login and registration system
- **Notifications**: Set reminders and notification preferences
- **Analytics**: View task completion statistics
- **Telegram Integration**: Connect with Telegram for notifications
- **Profile Management**: Customize user settings and profile
- **Dashboard**: Overview of tasks and progress
- **Responsive Design**: Works on desktop and mobile devices

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/SmartTo-Do_List.git
   ```

2. Navigate to the project directory:
   ```
   cd SmartTo-Do_List
   ```

3. Install dependencies (if any):
   ```
   pip install -r requirements.txt
   ```

4. Open the `Web/main_page.html` file in your web browser to start using the application.

## Usage

- **Home Page**: Start from `main_page.html` to explore the application
- **Registration/Login**: Create an account or log in to access personalized features
- **Dashboard**: View and manage your tasks from `dash_board.html`
- **Create Tasks**: Use `create_issue.html` to add new tasks
- **Settings**: Customize notifications, profile, and theme in the settings pages
